РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Информационная система «Автосалон»
1. Введение
1.1 Область применения
Данное руководство пользователя предназначено для работы с информационной системой «Автосалон», разработанной в среде Visual Studio. Система применяется для автоматизации учета автомобилей, покупателей и продаж в автосалоне.
1.2 Описание возможностей
Информационная система «Автосалон» предоставляет пользователю удобный интерфейс для выполнения следующих задач:
ведение базы данных автомобилей;
учет покупателей;
регистрация продаж;
поиск необходимой информации;
добавление новых записей;
сохранение изменений;
получение аналитических данных с помощью запросов.
Система позволяет быстро обрабатывать информацию и повышает эффективность работы автосалона.
1.3 Уровень подготовки пользователя
Для работы с программой пользователь должен:
владеть базовыми навыками работы с персональным компьютером;
уметь работать с окнами и меню;
иметь начальные навыки работы с приложениями Windows.
Специальной подготовки не требуется.
1.4 Перечень эксплуатационной документации
К эксплуатационной документации относятся:
руководство пользователя;
описание программы;
техническое задание (при наличии).
2. Назначение и условия применения
2.1 Виды деятельности, функции
Система предназначена для выполнения следующих функций:
учет автомобилей;
учет покупателей;
учет продаж;
поиск данных;
добавление и редактирование информации;
анализ данных с помощью встроенных запросов.
2.2 Программные и аппаратные требования
Минимальные требования:
Программные:
Операционная система: Windows 7 и выше
Наличие установленной платформы .NET Framework
Аппаратные:
Процессор: от 1 ГГц
Оперативная память: от 2 ГБ
Свободное место на диске: от 100 МБ
3. Подготовка к работе
3.1 Состав дистрибутива
В состав дистрибутива входят:
исполняемый файл программы;
база данных;
вспомогательные файлы (при наличии).
3.2 Запуск системы
Для запуска программы необходимо:
Открыть папку с программой;
Запустить исполняемый файл;
Дождаться загрузки главного окна.
3.3 Проверка работоспособности системы
Дата проверки: 02.12.2025
Для проверки необходимо:
Запустить программу;
Открыть любой раздел (например, «Автомобили»);
Попробовать добавить запись;
Выполнить поиск;
Сохранить изменения.
Если все операции выполняются корректно — система работает исправно.
4. Описание операций
4.1.1 Наименование операций
Основные операции системы:
добавление записи;
поиск записи;
сохранение данных;
просмотр таблиц;
выполнение запросов.
4.1.2 Условия выполнения операций
Операции выполняются при:
запущенной программе;
корректной работе базы данных;
наличии прав пользователя на выполнение действий.
4.1.3 Подготовительные действия
Перед началом работы необходимо:
Запустить программу;
Выбрать нужный раздел в меню:
«Таблицы»
«Запросы»
4.1.4 Основные действия
Работа с меню:
«Файл» → выход из программы
«Таблицы» → переход к:
автомобили
покупатели
продажи
«Запросы» → переход к:
самая популярная модель этого месяца
покупатель с наибольшей суммой покупки
Работа с таблицами:
Кнопка «Добавить»:
Нажать «Добавить»
Ввести данные
Нажать «Добавить» или «Закрыть»
Кнопка «Найти»:
Нажать «Найти»
Ввести критерий
Нажать «Найти» или «Закрыть»
Кнопка «Сохранить»:
сохраняет все изменения в таблице
4.1.5 Заключительные действия
После завершения работы:
Сохранить данные;
Закрыть открытые формы;
Выйти через меню «Файл» → «Выход».
4.1.6 Ресурсы, расходуемые на операции
Для работы системы используются:
оперативная память;
процессорные ресурсы;
дисковое пространство для хранения базы данных.
5. Аварийные ситуации
Возможные проблемы:
программа не запускается;
не сохраняются данные;
ошибка при выполнении операций.
Способы решения:
перезапустить программу;
проверить наличие базы данных;
убедиться в корректности введенных данных;
обратиться к разработчику.
6. Рекомендации по освоению
Для эффективного освоения программы рекомендуется:
последовательно изучить разделы меню;
протестировать функции добавления и поиска;
регулярно сохранять данные;
использовать запросы для анализа информации.






4. Описание операций
4.1 Операция «Добавление записи»
4.1.1 Наименование операции
Добавление новой записи в таблицы «Автомобили», «Покупатели», «Продажи».
4.1.2 Условия выполнения операции
Программа запущена;
Открыта одна из таблиц;
База данных доступна для записи.
4.1.3 Подготовительные действия
Запустить программу;
Перейти в раздел «Таблицы»;
Выбрать нужную таблицу (Автомобили / Покупатели / Продажи).
4.1.4 Основные действия
Нажать кнопку «Добавить»;
В открывшейся форме ввести данные;
Нажать кнопку:
«Добавить» — для сохранения записи;
«Закрыть» — для отмены операции.
4.1.5 Заключительные действия
Проверить, что запись появилась в таблице;
При необходимости нажать кнопку «Сохранить».
4.1.6 Ресурсы, расходуемые на операцию
Оперативная память;
Дисковое пространство для хранения данных.
4.2 Операция «Поиск записи»
4.2.1 Наименование операции
Поиск записи в таблицах системы.
4.2.2 Условия выполнения операции
Программа запущена;
Открыта таблица;
В таблице имеются данные.
4.2.3 Подготовительные действия
Открыть программу;
Перейти в раздел «Таблицы»;
Выбрать нужную таблицу.
4.2.4 Основные действия
Нажать кнопку «Найти»;
Ввести критерий поиска;
Нажать:
«Найти» — для выполнения поиска;
«Закрыть» — для выхода.
4.2.5 Заключительные действия
Просмотреть найденные результаты;
При необходимости повторить поиск.
4.2.6 Ресурсы, расходуемые на операцию
Оперативная память;
Процессорное время.
4.3 Операция «Сохранение данных»
4.3.1 Наименование операции
Сохранение изменений в базе данных.
4.3.2 Условия выполнения операции
Выполнены изменения в таблице;
Программа работает корректно.
4.3.3 Подготовительные действия
Выполнить добавление, редактирование или удаление данных.
4.3.4 Основные действия
Нажать кнопку «Сохранить»;
Дождаться завершения операции.
4.3.5 Заключительные действия
Убедиться, что изменения сохранены;
Продолжить работу или закрыть программу.
4.3.6 Ресурсы, расходуемые на операцию
Дисковое пространство;
Оперативная память.
4.4 Операция «Работа с запросами»
4.4.1 Наименование операции
Выполнение аналитических запросов.
4.4.2 Условия выполнения операции
Программа запущена;
В базе данных имеются записи.
4.4.3 Подготовительные действия
Открыть программу;
Перейти в раздел «Запросы».
4.4.4 Основные действия
Выбрать один из запросов:
«Самая популярная модель этого месяца»;
«Покупатель, купивший на самую большую сумму»;
Просмотреть результат.
4.4.5 Заключительные действия
Проанализировать полученные данные;
Закрыть окно запроса.
4.4.6 Ресурсы, расходуемые на операцию
Оперативная память;
Процессорное время.
4.5 Операция «Выход из программы»
4.5.1 Наименование операции
Завершение работы приложения.
4.5.2 Условия выполнения операции
Программа запущена.
4.5.3 Подготовительные действия
Сохранить все изменения.
4.5.4 Основные действия
Открыть меню «Файл»;
Выбрать «Выход».
4.5.5 Заключительные действия
Убедиться, что программа закрыта.
4.5.6 Ресурсы, расходуемые на операцию
Минимальные системные ресурсы.
