РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
Информационная система «Автосалон»
1. Назначение программы
Информационная система «Автосалон» разработана в среде Visual Studio и предназначена для автоматизации учета автомобилей, покупателей и продаж, а также для получения аналитической информации.
2. Запуск программы
Для начала работы необходимо:
Запустить приложение.
После запуска откроется главное окно программы с верхним меню (MenuStrip).
3. Главное меню программы
В верхней части окна расположено главное меню, включающее следующие разделы:
Файл
Таблицы
Запросы
4. Раздел «Файл»
Раздел «Файл» содержит команду:
Выход — завершение работы программы.
Для выхода из программы необходимо:
Нажать пункт меню «Файл»
Выбрать команду «Выход»
5. Раздел «Таблицы»
Раздел «Таблицы» предназначен для работы с основными данными системы.
Доступны следующие таблицы:
Автомобили
Покупатели
Продажи
6. Работа с таблицами
Для каждой таблицы предусмотрен следующий набор кнопок:
6.1 Добавление записи
Кнопка «Добавить» предназначена для внесения новой записи.
Порядок работы:
Нажать кнопку «Добавить»
В открывшейся форме ввести необходимые данные
Нажать одну из кнопок:
«Добавить» — для сохранения записи
«Закрыть» — для отмены операции
6.2 Поиск записи
Кнопка «Найти» позволяет найти необходимую запись.
Порядок работы:
Нажать кнопку «Найти»
Ввести критерий поиска
Нажать:
«Найти» — для выполнения поиска
«Закрыть» — для выхода из формы
6.3 Сохранение данных
Кнопка «Сохранить» является общей и используется для сохранения всех внесенных изменений.
Рекомендуется использовать кнопку после:
добавления записей
редактирования данных
удаления записей
7. Описание таблиц
7.1 Таблица «Автомобили»
Содержит информацию об автомобилях:
марка
модель
цена
характеристики
7.2 Таблица «Покупатели»
Содержит информацию о клиентах:
ФИО
контактные данные
7.3 Таблица «Продажи»
Содержит информацию о продажах:
автомобиль
покупатель
дата покупки
сумма сделки
8. Раздел «Запросы»
Раздел «Запросы» предназначен для получения аналитической информации.
8.1 Самая популярная модель этого месяца
Отображает автомобиль, который был продан наибольшее количество раз за текущий месяц.
8.2 Покупатель, купивший на самую большую сумму
Отображает клиента, совершившего покупку на максимальную сумму.
9. Навигация по программе
Для перехода между разделами необходимо:
Выбрать пункт в верхнем меню
Нажать нужный подраздел
10. Завершение работы
Для корректного завершения работы программы необходимо:
выбрать пункт меню «Файл» → «Выход»
11. Рекомендации пользователю
Регулярно сохраняйте изменения с помощью кнопки «Сохранить»
Внимательно проверяйте вводимые данные
Используйте функцию поиска для быстрого доступа к информации
