2. Основные подразделения
2.1 Административный блок
Администратор / ресепшн
Встреча клиентов

Прием заявок на симулятор
2.2 Технический блок
Техник / IT-специалист
Настройка и обслуживание ПК и симулятора

Мониторинг оборудования
2.3 Маркетинг и продажи (может совмещаться с директором)

Ведение кассы и учёта
Оплата налогов
Контроль расходов
4. Особенности для ПК-клуба с симулятором
Вначале может быть 2–3 человека: директор совмещает маркетинг и финансы, администратор контролирует зал, техник обслуживает оборудование.
В дальнейшем, при расширении, появляются отдельные специалисты по маркетингу, обслуживанию симуляторов и клиентскому сервису.
Важна гибкость: на старте одна роль может включать несколько функций.
